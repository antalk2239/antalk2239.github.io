使用selenium对拉钩网进行爬取

准备工作

  • 安装chrome
  • 安装chrome driver（选择与chrome对应的版本）
  • pip install selenium安装selenium
  • pip install lxml安装lxml
  • pip install re安装re

爬取思路

​ 拉钩网的反爬虫机制很恶心，要想顺利的爬取数据要做到一下几点：

  1. 控制爬虫的速度，速度过快会弹出登陆界面
  2. 在第一次访问时会有一个悬浮在整个页面上的一个广告，后面就没有了
  3. 他的数据的用js加载的，整个网页并没有刷新
